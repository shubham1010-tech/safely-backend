# Safely Backend

AI-powered scam awareness platform backend built using Node.js and Express.

## Features
- Phishing detection
- OTP fraud detection
- Scam keyword analysis
- REST API support

## Tech Stack
- Node.js
- Express.js
- MongoDB

## Installation

```bash
npm install
npm start
```

## API Base URL

```bash
http://localhost:8080
```
